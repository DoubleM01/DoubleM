Project Details

The Detection of Spine Deformities project leverages machine learning to diagnose lumbar spine deformities. It uses tools like Python, Roboflow API, and OpenCV to process medical images and provide accurate predictions.

This project demonstrates the integration of advanced deep learning models with real-world applications in the medical field.
