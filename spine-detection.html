Detection of Spine Deformities

Project Overview

This project focuses on leveraging deep learning models to analyze lumbar spine MRI images. The goal is to detect abnormalities such as disc herniation by localizing vertebrae, segmenting them, and analyzing intervertebral disc spacing. The system incorporates advanced image processing and machine learning techniques, including YOLO for vertebrae localization and TensorFlow for segmentation tasks.

Key Features

  • Localization of lumbar spine vertebrae (L1-L5).
  • Segmentation and analysis of intervertebral discs.
  • Detection of abnormalities like disc herniation.
  • Integration with Roboflow API for dataset processing.
  • Utilization of YOLO for high-accuracy object detection.

Tools and Technologies

  • YOLO v7 for pose estimation key points detection.
  • TensorFlow for deep learning models.
  • Roboflow for image preprocessing and annotation.
  • OpenCV for image processing tasks.
