Virtual Fitting Room

A Cutting-Edge Solution Bridging Technology and Fashion

About the Project

The Virtual Fitting Room project integrates computer vision, machine learning, and 3D modeling to revolutionize online shopping. Customers can visualize how clothes fit and look on their virtual models by uploading a video, enhancing the shopping experience.

Key Features

  • 3D Human Reconstruction using SMPL and EasyMocap
  • Pose Estimation using YOLO v7 for keypoint detection
  • Body Measurements for accurate sizing
  • Integration with NVIDIA Jetson platform for edge computing
  • Clothing simulation with realistic fabric behavior

Tools and Technologies

This project utilized a range of tools and platforms:

  • YOLO v7 for pose estimation
  • SMPL for 3D body modeling
  • NVIDIA Jetson Orin Nano for edge device compatibility
  • MediaPipe for body measurements
  • Python, TensorFlow, and OpenCV for machine learning and image processing
