Virtual Fitting Room

Revolutionizing Online Shopping with Advanced 3D Modeling and Simulation

Objective

To create an advanced Virtual Fitting Room (VFR) that allows customers to visualize how clothing fits on their bodies by generating a 3D model based on video inputs. This aims to enhance the online shopping experience and address common issues like inaccurate fit, high return rates, and the inability to physically try on clothing.

Key Features and Components

  • Pose Estimation & Body Measurements: Use of tools like OpenPose and SMPL for detecting body keypoints and generating a 3D human model. Extract accurate body dimensions such as chest, waist, and inseam from videos.
  • 3D Reconstruction: Utilize SMPL models for realistic human shape and pose creation. Integrate EasyMocap for aligning pose estimation data with the SMPL model.
  • Clothing Simulation: Simulate fabric behavior (elasticity, flow, fit) using physics-based algorithms.
  • Embedded Systems: ESP32 microcontroller is used to connect hardware (camera, sensors) to a mobile app.
  • Mobile Application: Developed in Android Studio, offering an intuitive interface for users to upload videos, view results, and store their profiles securely.
  • Socket Server and Database: A server (powered by NVIDIA Jetson Orin Nano) handles the processing and connects with a lightweight SQLite database for user authentication.

Advantages

  • Convenience & Safety: Allows customers to virtually try on clothes without removing their physical garments.
  • Increased Conversion Rates: Enhances customer confidence in purchasing decisions, reducing return rates.
  • Pandemic-Proof Shopping: Addresses hygiene and distancing concerns from fitting rooms.
  • Cost Savings for Retailers: Eliminates the need for physical fitting rooms and staff overheads.

Future Goals

  • Augmented Reality Integration: Enable users to visualize clothes on themselves in real-time via smartphone cameras.
  • Enhanced Simulation: Incorporate dynamic animations and fabric-specific physics.
  • AI-Powered Recommendations: Provide style and size recommendations based on user preferences.

System Workflow

The virtual fitting room operates with the following workflow:

  1. User Interaction: Upload a video or scan themselves using a mobile app.
  2. Data Processing: Pose estimation detects body landmarks, and body measurements are extracted for 3D modeling.
  3. Clothing Simulation: The chosen clothing is overlaid onto the 3D model, simulating fit and fabric behavior.
  4. Preview and Feedback: Users view a 360-degree model and receive recommendations for adjustments or alternative fits.

Workflow Chart: Input Video → Pose Estimation → 3D Reconstruction → Clothing Overlay → User Preview

Hardware & Software Stack

Hardware:

  • Camera or smartphone for video capture.
  • ESP32 microcontroller for embedded systems.
  • NVIDIA Jetson Orin Nano for server processing.

Software:

  • OpenPose, SMPL for pose estimation and 3D modeling.
  • Android Studio for mobile application development.
  • SQLite for secure database storage.
