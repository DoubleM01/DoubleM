About the Project

**Find My Game (FMG)** is a mobile computing project that simplifies the process of finding the best deals on video games across multiple platforms, including PC, PlayStation, Xbox, and Nintendo Switch. The application allows gamers to save time and money by consolidating game deal searches into a single platform.

Collaborators: Omar, Mahmoud, Ezzeldin, Mohamed, Karim, Salma

Features

  • Search for Game Deals: Simultaneously query multiple online stores for the lowest prices across platforms.
  • View Search Results: Display game title, price, and store provider for each platform in an easy-to-read format.
  • History Tracking: Track past searches, including game titles and platforms, for future reference.
  • Wishlist Management: Create and manage a wishlist linked to the user account.
  • Notification System: Get platform-specific notifications about game deals and discounts.
  • Currency Conversion: Automatically convert prices to EGP for stores that use USD.

Target Users

The primary users are gamers who actively seek the best deals on video games across various platforms. These users value efficiency, convenience, and cost-effectiveness in their gaming purchases.
